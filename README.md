# winkey95
Only for education and experimentation purpose. Please avoid piracy.

# How it works
Windows 95 key is formatted like this : "xxx-xxxxxxx". 
The first 3 digits don't matter at all. If the sum last 7 digits is divisible by 7, then it's a valid Windows 95 key.

# How to use it
1. Download the latest release.
2. Run the program.
3. Click "Generate!" button.
4. The program will show the key that you can select and then copy.
5. You can generate another key by pressing the "Generate!" button again.
