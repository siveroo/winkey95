# winkey95
Only for education and experimentation purpose. Please avoid piracy.

# How it works
Windows 95 key is formatted like this : "xxx-xxxxxxx". 
The first 3 digits don't matter at all. If the sum last 7 digits is divisible by 7, then it's a valid Windows 95 key.

# How to use it
1. Install Python (3.x version is recommended)
2. Download the .py file
3. Open the .py file
4. Type how many keys you want to be generated, and then press ENTER.
5. Done.
